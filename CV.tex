% --- 第一个项目 ---
\textbf{MiniOneRec - 端到端生成式推荐系统框架 (个人核心 / 开源项目)} \hfill 核心算法开发 \titlebreak
\textbf{项目背景}：针对传统推荐系统冷启动及多模态语义融合痛点，解决直接引入 LLM 存在大词表导致生成效率低下和推荐“幻觉”的问题。
\begin{itemize}[nosep, topsep=\intraProjectListTopSep]
    \item \textbf{表征构建}：应用多级 RQ-VAE 算法量化百万级候选物品，构建层级化语义 ID (SID)，缓解特征稀疏性；引入 Accelerate 多 GPU 推理，\textbf{效率提升 300\%}。
    \item \textbf{SFT 微调}：将用户交互历史转化为 Token，利用 Qwen2.5 进行序列推荐微调；引入价值感知微调 (VAFT)，使大模型外部知识与离散 ID 高效对齐。
    \item \textbf{RL 对齐}：设计基于 \textbf{GRPO} 的轻量级强化学习，融合排序置信度与 CF 分数构建混合奖励信号稳定梯度，有效打破历史曝光偏差。
    \item \textbf{项目成果}：开发 LogitProcessor 实施 Trie 约束解码，\textbf{无效生成降至 0}。在 Amazon 数据集上，相比基线模型，\textbf{HR@10 提升 15.6\%，NDCG@10 提升 18.2\%}。
\end{itemize}

\textbf{面向 OTTO 电商平台的大规模用户行为预测} \hfill 算法与特征工程 \titlebreak
\textbf{项目背景}：在 1.64 亿条交互日志、1100 万+ 匿名会话的数据集上，预测用户的点击、加购、下单行为并输出 Top-20 推荐。
\begin{itemize}[nosep, topsep=\intraProjectListTopSep]
    \item \textbf{多路召回}：构建 20 种共访矩阵，引入指数时间衰减权重 $w=(1/2)^{|\Delta t|/3600}$，为每位用户高效召回 100 个候选商品。
    \item \textbf{特征与排序}：构建 100+ 维特征；训练 Click/Cart/Order 三个 \textbf{XGBoost} 模型，采用负样本下采样与严格时间切分避免数据泄漏。
    \item \textbf{工程落地}：基于 \textbf{RAPIDS cuDF} 实现 GPU 加速，\textbf{计算性能提升 30 倍}，跑通端到端亿级流水线。融合 XGBoost 与 CatBoost，在 Kaggle \textbf{斩获 0.604 高分}。
\end{itemize}
